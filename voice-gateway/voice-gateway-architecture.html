Voice Gateway Architecture

Referencable Memory Snapshots — Layered RAG for Sub-Second Voice Responses

Snapshot Build

  • • Auto-generated from MEMORY.md + USER.md
  • • § entries → index + compressed summary
  • • Rebuilds on memory write or session start
  • • ~80 tokens, always in system prompt

EXPAND Protocol

  • • Snapshot lists entry refs: [0]-[16]
  • • LLM emits EXPAND[3,7] for detail
  • • Gateway injects full entries inline
  • • Follow-up turns keep expanded context

RAG Enrichment

  • • Keyword overlap scoring on § entries
  • • Top-k entries appended to prompt
  • • ~200 tokens total for most queries
  • • Zero overhead when no match

Model Routing

  • • Fast: Flash + snapshot (simple Q)
  • • Standard: 4.1-mini + RAG (memory)
  • • Deep: Full agent (tools, GitHub)
  • • Auto-escalate, async interject
